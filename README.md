📜 Glaze Engine Official README
Version: 1.04 "Gold"

Engine: Glaze Runner (Electron)

Language: Gluo1.04 (.g1)

1. Overview
The Glaze Engine is a high-performance 2D development environment. It uses Gluo1.04, a property-first scripting language that allows developers to separate object appearance from physical behavior.

2. Core Features
Property-Based Logic: Define player and world styles using the Local: prefix.

Build System: Create complex objects (Walls, Hazards, Wins) with a single line of code.

Geometry Engine: Native support for Squares (sq), Circles (cir), and Triangles (tri).

Level Management: Built-in support for the Level:[n] header to create multi-stage games.

3. IDE Controls
Run (.g1): Compiles the current script and starts Level 1.

Save/Open: Manage your .g1 files via the native OS file explorer.

Console: Real-time feedback for Print out commands and engine status.
